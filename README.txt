Recordar instalar los requeriments escritos en el archivo requeriments.txt

Para ejecutar la IA se debe iniciar el archivo run_web.bat, este abrira una pagina demo
la cual trae consigo 3 carritos precargados (los productos seleccionados pueden ser modificados desde el codigo).

Recordar que la idea de este proyecto es mostrar la funcionalidad de un agente IA en el entorno de cuellos de botella,
compatibilidad de productos, etc.

El archivo run_ai.bat ejecutara la ia sin un entorno usable, solamente es para beta y pruebas de codigo.
